# dio-desafio-projeto-mario-kart

Este repositório contém o projeto do desafio "Mario Kart" desenvolvido durante a jornada da DIO.

Conteúdo da pasta:

- `package.json`: arquivo de configuração do projeto Node.js, que inclui dependências e scripts de execução.
- `README.md`: documentação do projeto e explicação dos arquivos e da estrutura.
- `src/index.js`: arquivo principal da aplicação, onde está escrito o código JavaScript inicial do projeto.

Este projeto serve como base para desenvolvimento em Node.js e React, com foco na estruturação de código e no entendimento dos conceitos trabalhados durante o desafio.
